# Sentiment Analysis using Python

## Project Overview

This project was completed as part of the CodeAlpha Data Analytics Internship.

The objective of this project is to analyze customer reviews and classify them as Positive, Negative, or Neutral using Natural Language Processing (NLP).

## Tools Used

- Python
- Pandas
- TextBlob
- Matplotlib
- Google Colab

## Analysis Performed

- Text Processing
- Sentiment Classification
- Data Visualization

## Key Findings

- Positive reviews were the majority.
- Negative reviews identified dissatisfaction.
- Neutral reviews represented balanced opinions.

## Conclusion

Sentiment Analysis is a powerful NLP technique that helps businesses understand customer opinions and make better decisions.
